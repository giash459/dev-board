Question-1: What are the different ways to select an element in the DOM?

The ways to select an element in the DOM are : getElementById (ID), getElementsByClassName (class name), getElementsByTagName (tag name), querySelector (), and querySelectorAll ()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML কোডসহ ভেতরের কনটেন্ট পরিবর্তন করে, innerText শুধুমাত্র টেক্সট কনটেন্ট পরিবর্তন করে, textContent সমস্ত টেক্সট কনটেন্ট পরিবর্তন করে

Question-3: What is event delegation in the DOM?

ইভেন্ট ডেলিগেশন হল জাভাস্ক্রিপ্টের একটি কাজ যেটি একটি parent element একাধিক child elements পরিচালনা করে

Question-4: What is event bubbling in the DOM?

event bubbling একটি একক ইভেন্ট লিসেনার সেট করার অনুমতি দেয় যাতে তার চাইল্ড উপাদান থেকে ইভেন্টগুলি পরিচালনা করা যায়

Question-5: How do you create, add, and remove elements using JavaScript?

document.createElement("tagName") ব্যবহার করে একটি নতুন উপাদান তৈরি করা হয় , add element ব্যবহার করে একাধিক কাজ add করা হয় , remove element ব্যবহার করে কোন কিছু ডিলিট করা হয়
